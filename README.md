# Personal Portfolio - Marco Di Palma

This is my professional portfolio website, designed to showcase my journey as a Software Engineering student, my technical skills, and my personal projects.

## URL
https://portfolio-dipalma.netlify.app/#projects

## 🚀 Tech Stack
- **Framework:** React.js / Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (o Lucide React per le icone)
- **Deployment:** Netlify

## ✨ Key Features
- **Responsive Design:** Optimized for all devices (including iPhone 16 Pro).
- **Smooth Scrolling:** Enhanced UX for seamless navigation.
- **Multilingual Support:** Switch between English and Italian.
- **Project Showcase:** Direct links to my GitHub repositories and live websites.

## 🛠️ My Setup
